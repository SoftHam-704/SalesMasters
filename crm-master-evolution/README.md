# CRM Master - Evolution (Greenfield)

## Objective
The next level of Commercial Representation CRM, focusing on intelligent automation and lead hunting.

## Core Features
1. **Intelligent Pipeline**: Automatically moving opportunities based on BI data.
2. **Lead Hunter**: Automated prospecting using AI to search for companies and initiate WhatsApp contact.
3. **Sales Narratives (Gemini)**: Pre-visit reports that suggest the best approach for each client.
4. **Social Media & WhatsApp Integration**: Unified lead stream from multiple channels.
