# BI-Intelligence - Greenfield

## Objective
A modern, AI-powered Business Intelligence system built from scratch for SalesMasters.

## Architecture
- **Data Layer**: Clean PostgreSQL views and stored procedures (`fn_greenfield_*`).
- **Intelligence Layer**: Gemini 2.0 Pro / Flash for deep data analysis and predictive insights.
- **Frontend**: High-performance, Glassmorphism-based dashboard.

## Phase 1: Foundation
- [ ] Define standardized SQL views for Sales, Churn, and Growth.
- [ ] Implement Gemini Service for "Natural Language to SQL" (NL2SQL).
- [ ] Create `BiGreenfield.jsx` page with modern charts.

## Phase 2: AI Automation
- [ ] Automated Sales Narratives.
- [ ] Product Correlation Engine (Market Basket Analysis).
- [ ] Automated Goal Adjustment based on trends.
